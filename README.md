# README

## Purpose:
This repo is my course project for the *Getting and cleaning data* Coursera course. 
The program run_analysis.R produces means and standard deviations for a dataset of subjects performing various activities while the subjects' Samsung smartphones are recording angular velocity and acceleration data. 

The data source is at https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip. The set is documented at http://archive.ics.uci.edu/ml/datasets/Human+Activity+Recognition+Using+Smartphones. 

run_analysis.R performs the following steps:
* If there is no "UCI HAR Dataset" directory, it downloads and unzips the data into that directory. 
* It reads the source features, activities, and subject's results from both a training dataset and a test dataset.
* It merges the training and test datasets.


## Codebook
* Subject: A numeric ID identifying the subject
* Activity: A description of the activity the subject was performing
** WALKING: subject was walking
** WALKING_UPSTAIRS: subject was walking upward
** WALKING_DOWNSTAIRS: subject walking downward
** SITTING: subject was sitting 
** STANDING: subject was standing 
** LAYING: subject was laying down
